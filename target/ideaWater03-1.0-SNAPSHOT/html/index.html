数据采集 更多

  • 定点采集

    在监测点安置监测仪器，监测仪器通过各个传感器模块，获取相应水质数据，并将其上传至服务器。

  • 无人机采集

    使用无人机航拍技术，获取河道表面最新图像信息，可以直观的看到河道整体情况，了解水质情况。

  • 移动监测

    在移动检测船上安置摄像头，实时查看检测点的水体情况，了解水体表面的实时信息以及仪器的工作状况。

  • 人工采集

    相关水质专业检测人员，不定期至河道各地段采集相关材料，考察该地段水质情况，并做相关记录。
